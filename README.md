# Import

Ez egy Eclipse + Maven projekt. Importáljátok Eclipse-be klónozás után.

# Build és tesztelés

 - Build: Jobb klikk a projektre -> Run as... -> Maven build
 
 - Tesztek: Jobb klikk a test mappára projekten belül -> Run as... -> JUnit test
